# FitDaddy - AI Powered Fitness Companion (2025)
---

## Project Overview
---
FitDaddy is a mobile-first AI fitness application that delivers personalized workout plans, diet recommendations, and real-time fitness guidance.

---

## Key Features
---
- Mobile-first AI fitness application
- Personalized workout and diet recommendations
- Conversational AI coach for real-time guidance
- CNN-based food recognition for calorie and macro estimation

---

## Tech Stack
---
- Frontend: Flutter
- Backend: Python (FastAPI)
- AI/ML: Recommendation Systems, CNN
- Database and Authentication: Firebase
